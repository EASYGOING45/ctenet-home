# PLAN.md — ctenet-home

## 项目定位

`ctenet-home` 是一个面向 AI 时代的个人站点。

它用来展示：
- 我是谁
- 我在做什么
- 我有哪些代表性项目
- 我如何思考与构建系统

它不是普通名片页，也不是概念口号页，而是一个可长期演进的个人数字入口。

---

## 功能清单

### V1 核心模块
- Hero：快速说明身份与方向
- About：解释我是谁、我关注什么
- Method：说明我的构建方式与系统思维
- Projects：展示代表项目与能力证据
- Now：展示当前重点与持续推进的方向
- Contact：提供真实出口（优先 GitHub）

### 当前不做
- 博客系统
- CMS / 后台
- 多语言
- 登录系统
- 复杂动态数据
- 过重炫技动画
